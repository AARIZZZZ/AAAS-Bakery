💳 Razorpay Payment Setup
Configure your Razorpay API key so customers can pay directly into your bank account. You only need to do this once.
⚠️ Not configured yet
Step 1 — Create a Razorpay Account
Go to razorpay.com → Sign Up (it's free). Use your business name AAAS Bakery.
Step 2 — Add Your Bank Account
In Razorpay Dashboard → Settings → Bank Account → Add your account. All payments go there automatically.
Step 3 — Get Your API Key
Dashboard → Settings → API Keys → Generate Live Key. It starts with rzp_live_. Paste it below.
This is stored in your browser only — never sent anywhere else.

📸 Instagram Photo Sync
Paste direct image URLs from your Instagram posts to instantly add them to the gallery. No API key needed.
1
Open Instagram on desktop
Go to @aaas_bakery_ and open the post you want to add
2
Copy the image URL
Right-click the photo → "Open image in new tab" → copy the URL from the address bar. It usually starts with https://scontent...
3
Paste below & add to gallery
Paste the URL, add a caption and category, then click Add. It'll appear in the gallery instantly.
💡
Pro tip
Instagram CDN URLs expire after ~24 hours. For permanent photos, download the image from Instagram, save it in the same folder as index.html, and use the filename instead of the URL.
